# Product Viewer

Een single-page React-applicatie waarmee gebruikers producten kunnen bekijken, filteren en details zien.

## Functionaliteit
- Producten laden via FakeStoreAPI
- Zoekbalk voor filtering
- Selecteerbaar product met detailweergave
- Tailwind CSS styling

## Installatie
```bash
npm install
npm start
```

## Builden
```bash
npm run build
```
